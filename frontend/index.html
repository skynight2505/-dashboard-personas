Instrucciones: Cómo organizar los documentos

1. Crea una carpeta raíz en Google Drive

Nombra la carpeta como quieras (ej: DashboardPersonas). Esta será la carpeta principal que el sistema leerá.

2. Dentro, crea una subcarpeta por cada categoría

Cada subcarpeta debe contener SOLO UNA CATEGORÍA. El sistema clasificará todos los archivos dentro de esa carpeta con la misma categoría:

📁 DashboardPersonas (carpeta raíz)
📁 Vivos → Vivo / Sitio actual
📁 Hospital → Vivo / Sitio actual
📁 Desaparecidos → Desaparecido
📁 Fallecidos → Fallecido

Los nombres de carpeta reconocidos: Vivos, Vivo, Hospital, Heridos, Desaparecidos, Desaparecido, Extraviados, Fallecidos, Fallecido, Muertos, Occisos

3. Cada archivo debe listar personas con nombre y cédula

Los documentos (PDF, Excel, Word) deben contener listas de personas. El sistema extraerá automáticamente nombre y cédula de cada registro.

Excel recomendado

CédulaNombre CompletoEdad
12345678Juan Pérez López35
87654321María García Rojas28

Las columnas pueden llamarse: cédula, cedula, identificación, documento, id y nombre, nombre_completo, nombres, apellido

PDF / Word

Puede tener formato de lista o tabla. El sistema buscará patrones como:

12345678 Juan Pérez López
87654321 María García Rojas

O en formato tabla dentro del documento.

4. Búsqueda por nombre o cédula

Una vez procesados los documentos, puedes buscar cualquier persona por nombre o cédula en la barra de búsqueda de abajo. El sistema encontrará la persona sin importar en qué documento o categoría fue registrada.

5. Reclasificación manual

Si una persona está en la categoría incorrecta, puedes editarla manualmente haciendo clic en el botón de editar.
